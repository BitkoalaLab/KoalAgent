The Proactive AI Agent
With a Beating Heart.

一个原生开源的个人专属 AI 智能体。拥有长久记忆、自主心跳机制，以及零配置的飞书双向通讯能力。

在 GitHub 上查看 探索特性
🧠

原生长期记忆

基于 Mem0 与 BGE 向量模型。它能精准提取你的身份画像与喜好，打破大模型单轮对话的遗忘曲线。

🫀

自主心跳引擎

基于调度器的后台守护进程。即使你不发消息，它也会根据设定的时间间隔主动苏醒，向你发送关怀与资讯。

🛠️

实体抓手

原生支持函数调用。遇到自己不懂的事情，它能自主调用本地 Python 工具获取物理世界的真实数据。

💬

飞书无缝接入

深度接入飞书长连接 WebSocket 模式。无需公网服务器或繁琐的 Webhook 配置，本地运行即可在手机端畅聊。

演进路线图

Phase 0: 记忆与心跳
